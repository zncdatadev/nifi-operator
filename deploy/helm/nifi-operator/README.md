# Operator Chart for Apache Hadoop Nifi on Kubedoop

This chart bootstraps an Apache Hadoop Nifi Operator deployment on a Kubernetes cluster using the Helm package manager. It's part of the Kubedoop ecosystem.

## Installing the Chart

To install the chart with the release name `nifi-operator`:

```bash
helm install nifi-operator oci://quay.io/kubedoopcharts/nifi-operator
```

## Usage

The operator example usage can be found in the [examples](https://github.com/zncdatadev/nifi-operator/tree/main/examples) directory.
